## 视图是虚拟的表，是SELECT返回的结果集

源表增加数据对视图没影响，修改列名可以通过修改视图解决

使用视图的用户只能访问被允许查询的结果集

### 语法

```sql
create [or replace]

view view_name[(column_list)] as [select_statement]

[with [CASCADED | LOCAL ] CHECK OPTION]

alter view view_name[(column_list)] as select_statement

drop view view_b
```

*show tables会查询出视图*
